# 슬랙에서 대화하면 거래명세서가 만들어진다고?

- **날짜**: 2026-02-21
- **트랙**: A (날것)
- **시리즈**: 거래명세서 자동화 (EP.01 / 전체 3편)
- **예상 길이**: 약 10분

## 5줄 브리프

1. **문제**: 거래명세서를 매번 엑셀로 수작업 → 양식 찾고, 숫자 채우고, PDF 변환하고... 건당 30분
2. **시도**: Claude Code로 슬랙 봇을 만들어서, 슬랙에서 LLM과 대화하면 거래명세서가 자동 생성되게
3. **과정**: 슬랙 앱 생성 → Claude Code로 대화형 데이터 수집 봇 작성 → HTML 템플릿으로 거래명세서 변환
4. **결과**: 슬랙에서 대화 몇 마디 → HTML 거래명세서 자동 완성 (30분 → 2분)
5. **한줄 회고**: 슬랙 앱 권한 설정에서 헤맸지만, Claude한테 에러 메시지 붙여넣기하니 바로 해결

## 이번 영상의 범위 (스코프)

이번 영상에서 **하는 것**:
- 슬랙 앱 생성 및 봇 설정
- LLM 대화로 거래 데이터 수집 (거래처명, 품목, 수량, 단가 등)
- 수집된 데이터를 HTML 템플릿으로 거래명세서 변환

이번 영상에서 **하지 않는 것** (다음 영상으로 넘김):
- 거래 데이터 영구 저장 (→ EP.02 Airtable)
- PDF 변환 및 발송 (→ EP.03)

**분리 이유**: 한 영상에 다 넣으면 20분 넘어감. "슬랙 대화 → 거래명세서" 하나만으로도 충분한 가치.
